## 📋 新增功能

### 🔑 API 配置管理
- 首页「API 配置」弹窗管理多套 API 配置，支持故障转移
- 配置保存到服务端 session，跨页面不丢失

### 📚 智能问题库
- 自动记录 AI 问答，相似问题 ≥80% 复用缓存答案，防 AI 幻觉，节约API费用
- 消息气泡显示「📚 问题库」标签，可选「坚持请求AI回答」
- 右侧面板随时查看所有已问问题

### 🤖 AI 生成海龟汤
- 房主工具箱一键生成故事
- 故事广场独立「AI生成」标签页
- 多轮对话微调 + 逐条撤回
- 提示词预设编辑器（多角色消息 / 导入导出 / 恢复默认）

### 🎮 游戏优化
- 串行提问：同一时间只处理一个问题
- 切换故事自动清空 AI 上下文和问题库
- 通关弹窗：回到房间 / 退出房间
- AI 主持人可判断问题重要性

## 🎮 使用说明

### 创建房间
1. 首页 → API 配置 → 添加 API Key
2. 创建房间 → 输入昵称
3. 工具箱 → AI 生成故事 / 从广场加载 / 上传题目
4. 玩家提问，AI 自动回答

### 故事广场
- **浏览**：搜索、筛选故事，一键开玩
- **发布**：上传 JSON 文件或在线创作（管理员审核后发布）
- **管理/编辑**：修改已发布的故事
- **AI 生成**：输入要求让 AI 生成故事，可对话微调

### 管理员后台

## 🔧 常见问题

**Q: 别人加入房间需要配置 API 吗？**
A: 不需要。加入房间只需昵称和邀请码。

**Q: 为什么提示"AI正在回答上一个问题"？**
A: 串行机制，同一时间只能处理一个问题，等当前回答完再试。

**Q: 刷新页面后 API 配置丢失？**
A: 检查浏览器 cookie 是否被清除。配置存储在服务端 session。

**Q: 审核流程是怎样的？**
A: 用户上传 → 待审核列表 → 管理员批准 → 故事广场可见。
